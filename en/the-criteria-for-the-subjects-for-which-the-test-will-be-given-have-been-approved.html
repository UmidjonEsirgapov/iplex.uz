The criteria for the subjects for which the test will be given have been approved

The latest news in the field of education is as follows:

Brief content:
According to it, the number of test tasks and evaluation criteria are mandatory for 3 subjects (mother tongue, mathematics and history of Uzbekistan): a total of 30 test tasks, 10 from each subject; for each correct answer - 1.1 points; maximum score - 33 points.
1- specialty subject: number of test tasks - 30; for each correct answer - 3.1 points; maximum score - 93 points.
2nd specialty subject: number of test tasks - 30; for each correct answer - 2.1 points; the maximum score is 63 points.

For the 2024-2025 academic year, the number of test assignments and evaluation criteria for the subjects for which the test will be given in accordance with the undergraduate educational directions of higher education institutions have been approved. This was announced yesterday, on March 31, by the Agency for Assessment of Knowledge and Skills.

According to it, the number of test tasks and evaluation criteria are mandatory for 3 subjects (mother tongue, mathematics and history of Uzbekistan):

  • a total of 30 test tasks of 10 from each subject;
  • each correct answer for - 1.1 points;
  • maximum score - 33 points.

1- specialty subject:

  • number of test tasks - 30;
  • for each correct answer - 3.1 points;
  • maximum score - 93 points.

In the 2nd specialization subject:

  • number of test tasks - 30;
  • for each correct answer - 2.1 points;
  • maximum score - 63 points.

Number of test tasks in 5 subjects in total - 90. The maximum possible score is 189 points.
