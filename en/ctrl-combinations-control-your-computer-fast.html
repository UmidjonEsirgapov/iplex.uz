CTRL combinations: Control your computer quickly!

Keyboard shortcuts (hotkeys) are important to improve computer efficiency. Especially the combinations involving the CTRL key help to perform many actions quickly.

Many people use the Ctrl+C and Ctrl+V combinations, but do you know other important shortcuts CTRL + A to Z? Below are their main functions:

Main functions of CTRL combinations

  • Ctrl + A – Select all text or files
  • Ctrl + B – Make text bold
  • Ctrl + C – Copy text or file
  • Ctrl + D – Duplicate (copy in PowerPoint create)
  • Ctrl + E – Center text
  • Ctrl + F – Search in text or file
  • Ctrl + G – Group (in PowerPoint)
  • Ctrl + H – Replace text (in Word)
  • Ctrl + I – Italicize text
    • Ctrl + J – Spread text across the page (in Word)
  • Ctrl + O – Open a document or file
  • Ctrl + P – Print (print)
  • Ctrl + Q – Exit the program
  • Ctrl + R – Place the text to the right
  • Ctrl + S – Save (save)
  • Ctrl + T – Open a new folder (Google Chrome, Edge). Z – Undo the last action
