What happens if I get the same score but the number of quota is less?

Important news for students and applicants has been announced.

Brief content:
If several applicants get the same score in the last line of the grant or contract, which of them is considered to have entered the grant or contract?
3 applicants in the 9th, 10th, 11th places have the same score collected.
In such a situation, all three will be accepted for the state grant, an additional place will be allocated for the state grant; the same situation in the contract.

If several applicants score the same points in the last line of the grant or contract, which one of them is considered to have entered the grant or contract?

ANSWER AS FOLLOWS: — More information

  1. suppose 10 places are allocated to the state grant. 3 applicants in the 9th, 10th, 11th places scored the same points. In such a situation, all three are accepted for the state grant, an additional place is allocated for the state grant;
  2. the same situation in the contract. At the end of the entry line, if the applicants who are more than the quota score the same points, they are all considered to be admitted and an additional place will be allocated;
  3. the same rule applies to both undergraduate and master's degrees.
