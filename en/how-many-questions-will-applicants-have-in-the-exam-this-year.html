How many questions will applicants have in the exam this year?

Important news for students and applicants has been announced.

Brief content:
For the 2024-2025 academic year, the number of test tasks and evaluation criteria for the subjects for which the test will be given in accordance with the undergraduate educational directions of HEIs have been approved.
The number of test tasks and evaluation criteria: Compulsory 3 subjects (main language, mathematics and history of Uzbekistan): a total of 30 test tasks, 10 from each subject; for each correct answer - 1.1 points; maximum score - 33 points.
1st specialty subject: number of test tasks - 30; for each correct answer - 3.1 points; maximum score - 93 points.

For the 2024-2025 academic year, the number of test tasks and evaluation criteria for the subjects in which the test will be given for the baccalaureate educational directions of higher education institutions have been approved.

The number of test tasks and evaluation criteria:

Compulsory 3 subjects (mother tongue, mathematics and history of Uzbekistan):

  • a total of 30 test items of 10 in each subject;
  • for each correct answer - 1.1 points;
  • maximum score - 33 points.

In the 1st specialty:

  • number of test questions - 30;
  • for each correct answer - 3.1 points;
  • maximum score - 93 points.

In the 2nd specialty:

  • test questions number - 30;
  • for each correct answer - 2.1 points;
  • maximum score - 63 points.

The total number of test tasks in 5 subjects - 90. The maximum possible score is 189 points.
