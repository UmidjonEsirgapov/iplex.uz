# iplex.uz

Static website for educational articles in multiple languages (English & Russian).

## Structure

```
/
├── index.html          # Landing page
├── sitemap.xml        # Sitemap for SEO
├── .htaccess          # Apache rewrite rules (clean URLs)
├── en/                # English articles
│   ├── index.html
│   └── *.html (3708 articles)
└── ru/                # Russian articles
    ├── index.html
    └── *.html (3708 articles)
```

## Features

- Multi-language support (EN/RU)
- Clean URLs (no .html extension)
- SEO optimized
- Internal linking strategy
- Responsive design with Pico.css

## Deployment

This repository contains the static website files ready for deployment.

### GitHub Pages

1. Push to GitHub
2. Enable GitHub Pages in repository settings
3. Select main branch as source

### Traditional Hosting

1. Upload all files to hosting root directory
2. Ensure `.htaccess` is uploaded (for Apache servers)
3. Submit `sitemap.xml` to Google Search Console

## URL Structure

- Landing: `https://iplex.uz/`
- English: `https://iplex.uz/en/`
- Russian: `https://iplex.uz/ru/`
