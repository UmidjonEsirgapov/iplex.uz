Начиная с 2024-2025 учебного года экзамены для поступления в государственные высшие и профессиональные учебные заведения (за исключением профессиональных училищ) будут проводиться одновременно в соответствии по принципу «сначала тест, потом отбор». С 5 июня по 25 июня (также в эти дни) она будет проходить онлайн через сайт my.uzbmb.uz.

На 1 этапе регистрации абитуриенты выбирают:

  • комплекс предметов (по);
  • язык обучения (все для);
  • алфавит тестирования (если в качестве языка обучения выбран узбекский язык);
  • поле тестирования (для).
  • профессиональный (творческий) вуз, в котором будет сдаваться экзамен (если выбрана область, требующая особых талантов).

В данном случае он выбирается для определения диапазона направление обучения не означает, что абитуриент зарегистрирован по данному направлению (направление обучения выбирается после тестирования). Он создан для удобства абитуриентов для определения круга предметов. Если абитуриент выбирает область обучения, требующую особых талантов, он может выбрать вуз, удобный для него для сдачи профессионального (творческого) экзамена (так же, как он выбирает направление тестирования).
