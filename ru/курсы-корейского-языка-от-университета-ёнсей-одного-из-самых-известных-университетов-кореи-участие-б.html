Курсы корейского языка от Университета Ёнсей, одного из самых известных университетов Кореи

Для кого? Для начинающих и для тех, кто хочет изучать разговорный корейский язык

Участие абсолютно бесплатное

Этот курс предназначен для новичков, знакомых с алфавитом хангыль корейского языка. С помощью этого курса студенты приобретут необходимые знания для повседневного общения на корейском языке.

Курс состоит из шести модулей, а каждый модуль состоит из пяти блоков. Каждый блок содержит лексику, грамматические правила и предложения, устную разговорную практику, видеоролики, викторины, рабочую тетрадь и словарь необходимых слов.
