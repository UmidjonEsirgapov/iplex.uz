Обновлен список именных стипендий для студентов

На сегодняшний день самыми актуальными новостями в сфере образования стали:

Краткое содержание:
Принят Указ Президента (10.08.2023, ПФ-132).
Согласно ему в Указ №ПФ-555 внесены следующие изменения. Государственные стипендии по следующим направлениям установлены для студентов двух последних курсов (далее - старших курсов), имеющих 25 процентов «хорошо») и занимающихся творческой деятельностью:
по инженерному делу - Беруни, по медицине - Ибн Сина, по гуманитарным наукам - Навои, изобразительному и прикладному искусству, по дизайну - Камолиддин Бехзад, по естественным направлениям - Улугбек, по кинематографии - Худойберган Девонова, по истории и географии - Бабур, по физкультуре и спорту - Пахлавон Махмуд, в Международной исламской академии Узбекистана - Имам аль-Бухари, в высших учебных заведениях Республики Каракалпакстан - Толепберген Кайпбергенов и Ибраим Юсупов, а по всем направлениям - Ислам Каримов.

Указ Президента (10.08.2023, ПФ-132) был принят на вооружение. Согласно ему в Указ №ПФ-555 внесены следующие изменения. Следующие изменения внесены для студентов последних двух курсов (далее - выпускной курс), имеющих показатели успеваемости по всем предметам на «отлично» (максимум 25% «хорошо») и занимающихся творческой деятельностью.

  • по технологии - Беруни,
  • по медицине - Ибн Сина,
  • по гуманитарным наукам - Навои,
  • по изобразительному и прикладному искусству и дизайну - Камолиддин Бехзод,
  • по естественным направлениям - Улугбек,
  • по кинематографии - Худойберган Девонов,
  • по истории и географии - Бабур,
  • Пахлавон Махмуд по физкультуре и спорту,
  • Имам аль-Бухари в Международной исламской академии Узбекистана,
  • Толепберген Кайпбергенов и Ибраим Юсупова в высших учебных заведениях Республики Каракалпакстан,
  • также по всем специальностям - Ислама Каримова.

Указано, что: — Официальные данные

  • Государственные стипендии им. Беруни, Ибн Сины, Навои, Камолиддина Бехзада и Улугбека - по одной на каждую тысячу студентов высших курсов довузовского образования;
  • Государственная стипендия им. Имама аль-Бухари для трех студентов старших курсов Международной исламской академии Узбекистана;
  • Все остальные известные стипендии, присуждаемые каждому вузу, должны быть вручены одному студенту учебного заведения.
