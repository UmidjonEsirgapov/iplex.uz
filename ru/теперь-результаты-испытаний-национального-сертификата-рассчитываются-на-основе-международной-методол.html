Внесены изменения в постановление правительства «О внедрении национальной тестовой системы оценки уровня знаний по общеобразовательным предметам».

Согласно ему осуществляется регистрация претендентов на сдачу национальных аттестационных экзаменов по общеобразовательным предметам. осуществляется через Единый портал интерактивных государственных услуг (сайт my.gov.uz).

90 процентов установленной платы взимается с заявителя, поскольку регистрация для участия в национальном аттестационном экзамене осуществляется через сайт my.gov.uz.

Лицам, зарегистрированным в информационной системе «Единого реестра социальной защиты», также 50-процентная скидка распространяется на лиц с ограниченными возможностями I и II групп.

Результаты тестирования рассчитываются по международной методике – Rash модель. По нему:

Оценка В+ для ученика 60-64,9 класса;

Борт для ученика 55-59,9 класса;

Б Оценка для ученика 50-54,9 класса

С+ уровень;

Абитуриенту, получившему 46 – 49,9 баллов – уровень C.

Абитуриенту, получившему сертификат уровня A+, A, на вступительных испытаниях на степень бакалавра высших учебных заведений будет присвоен максимальный балл по данному предмету.

Абитуриенту, получившему сертификат уровня B+, B, C+, C, будут начислены пропорциональные баллы относительно максимального балла. присуждается наивысший балл. — Источник
