Организован курс по искусственному интеллекту и робототехнике для школьников с участием корейских специалистов

Создан курс по искусственному интеллекту и робототехнике для учащихся школы 129 Учтепинского района. В этом курсе 32 ученика выпускного класса школы изучают искусственный интеллект и робототехнику у корейских специалистов.

Для курса На основе по гранту Министерства образования Кореи сформирована материально-техническая база лаборатории робототехники. Для этой лаборатории были привезены Комплект Arduino и робот PIBO. С помощью этих устройств студенты получат возможность проверить свои теоретические знания по программированию на роботизированных устройствах. Планируется распространить положительный опыт на другие школы. — Официальные данные
