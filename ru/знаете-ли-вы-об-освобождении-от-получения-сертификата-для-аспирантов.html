Знаете ли вы об освобождении от получения сертификата для аспирантов?

Объявлены важные новости для студентов и абитуриентов.

Краткое содержание:
Преференции для лиц с ограниченными возможностями при поступлении в магистратуру, в Республике Узбекистан создаются условия для лиц с ограниченными возможностями.
Для кого это касается?

Преференции для людей с ограниченными возможностями при поступлении в магистратуру

В Республике Узбекистан создаются учебные заведения для людей с ограниченными возможностями. В частности, введена новая процедура, согласно которой для поступления в магистратуру не требуется сертификат по иностранному языку.

Для кого?

Эта льгота предназначена для лиц с нарушениями зрения, слуха и речи. Такие лица освобождаются от предъявления сертификата об иностранном языке при поступлении в государственные высшие учебные заведения:

  • магистратура;
  • специальности послевузовского образования.

Примечание

Данная процедура предназначена не для всех лиц с ограниченными возможностями. Пользоваться данной льготой имеют право только лица, относящиеся к трем вышеуказанным категориям.

Правовая основа
