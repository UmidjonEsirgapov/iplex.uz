Когда начнутся вступительные экзамены в спортивные школы имени Бахадира Джалолова и Оксаны Чусовитиной?

Объявлены важные новости для студентов и абитуриентов.

Краткое содержание:
Первый тур вступительных экзаменов в спортивную школу по боксу имени Бахадыра Джалолова и спортивную школу по спортивной гимнастике имени Оксаны Чусовитиной пройдут 27-29 августа текущего года.
Бокс имени Бахадыра Джалолова Для абитуриентов, поступающих в специализированную спортивную школу, Экзамен будет проводиться по адресу: Государственный университет физической культуры и спорта Узбекистана (Ташкентская область, город Чирчик, улица «Спортчилар», 19). школа (г.Ташкент, Мирабадский район, улица Сарыкол, пункт назначения: 175 общеобразовательная школа).

Специализированная спортивная школа бокса имени Баходира Джалолова и специализированная спортивная школа гимнастики Оксаны Чусовитиной, первый тур вступительных экзаменов в этом году 27-29 августа

Экзамен для абитуриентов, подавших документы в спортивную специализированную школу по боксу имени Бахадира Джалолова будет проводиться по адресу:

  • Узбекский государственный университет физического воспитания и спорта
  • (Ташкентская область, город Чирчик, улица «Спортчилар», дом 19).

Специализированная спортивная школа по спортивной гимнастике Оксаны Чусовитиной будет проводиться по следующему адресу:

  • Специализированная спортивная школа по спортивной гимнастике Оксаны Чусовитиной
  • (город Ташкент, Мирабадский район, улица Сарыкоул, адрес: 175-общеобразовательная школа).
