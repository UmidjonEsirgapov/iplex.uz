Допуск к Олимпийским играм без экзамена: какие победители олимпиад имеют право?

Полезная информация для абитуриентов и студентов:

Резюме:
Право на поступление в ОТМ без экзамена имеют победители следующих 5-ти международных олимпиад: 1.
Заключение В ОТМ Узбекистана без экзамена допускаются только победители международного этапа олимпиад IMO, IPhO, IChO, IBO и IOI.
Предпочтения другим победителям олимпиад отдаются только в виде максимального балла по предмету специальности.

Поступление в ОТМ без экзамена: какие возможности есть у победителей олимпиад?

Согласно Указу Президента № PQ-4306 от 3 мая 2019 года некоторые международные олимпиады победители принимаются в государственные высшие учебные заведения Узбекистана. принимаются без зачета и дополнительного экзамена. Эта привилегия введена в целях поддержки перспективной молодежи и поощрения талантливых студентов.

На каких международных олимпиадах предоставляется льгота?

Победители следующих 5 международных олимпиад имеют право поступить на ОТМ без экзамена:

Победители олимпиад, включенные в этот список, принимаются в качестве студентов напрямую без сдачи зачета в высшие учебные заведения.

Каковы привилегии других победителей олимпиад?

За исключением 5 перечисленных выше олимпиад, ни одна международная или местный не дает права на поступление в ОТМ без экзамена.

Для победителей Республиканской олимпиады естествознания:

  • Призерам 1, 2, 3 места начисляются максимальные баллы по предмету специальности.

  • Эти баллы действительны в течение 3 лет делает.

  • Например:

    • Лауреат по химии - если это основной предмет -> 93 балла

    • При наличии второго предмета по специальности -> 63 балла

По остальным предметам абитуриент сдает тестирование самостоятельно.

Особый порядок прохождения предмета ИТ

  • Победители республиканской олимпиады по предмету «Информатика и информационные технологии» имеют право на получение максимального балла в предмет математика.

Заключение

В вузы Узбекистана без экзамена будут допущены только те, кто победил на международном этапе олимпиад IMO, IPhO, IChO, IBO и IOI. Победителям других олимпиад льгота предоставляется только в виде максимального балла по специальности.
