Если абитуриент, готовящийся к изучению родного языка как обязательного предмета (к 10-балльному тесту), получит результат выше 60 процентов на национальном аттестационном экзамене, ему будет присвоен максимальный балл. (11 баллов) по обязательному родному языку.

  • Если абитуриент наберет 86% и выше, на вступительных экзаменах по соответствующему предмету будет выставлена ​​максимальная оценка. То есть 93 балла по родному языку и литературе как предмету 1-го блока, 63 балла по предмету 2-го блока. Также выставляется максимальная оценка (11 баллов) по 10 обязательным тестам.

  • Если абитуриент, который готовится к обязательному предмету (к 10-балльному тесту) на своем родном языке, и получает результат выше 60% на национальном аттестационном экзамене, ему будет присвоено максимальное количество баллов (11 баллов) по его обязательному родному языку. То есть национальный аттестат действителен и по обязательным предметам!
