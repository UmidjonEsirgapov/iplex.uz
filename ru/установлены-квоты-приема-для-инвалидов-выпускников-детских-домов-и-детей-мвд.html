Установлены квоты приема для инвалидов, выпускников детских домов и детей МВД

На сегодняшний день самыми актуальными новостями в сфере образования стали:

Краткое содержание:
Заявлением Государственной комиссии №1 утверждено распределение приема на основе дополнительного 2% государственного гранта для лиц с ограниченными возможностями здоровья в разделе высшие учебные заведения и формы обучения. Нормы приема на основе дополнительного 1% государственного гранта для детей-сирот распределяются по высшим учебным заведениям и формам обучения.
Согласно этому, всего по республике выделено 875 дополнительных квот для детей сотрудников органов внутренних дел.

Для лиц с ограниченными возможностями, выпускников Дома милосердия и Детского городка, детей сотрудников органов внутренних дел установлены дополнительные нормы приема утверждено.

Распределение показателей приема на основе дополнительного 2-процентного государственного гранта для лиц с ограниченными возможностями по разделу высшие учебные заведения и формы обучения утверждено постановлением Государственной комиссии № 1.

Согласно ему, всего для лиц с ограниченными возможностями по республике выделено 3155 дополнительных квот. По формам обучения:

  • по дневному обучению - 2270;
  • по экстернату - 754;
  • по вечернему обучению - 131.

Прием на основе дополнительного 1% государственного гранта для реальных детей-сирот, окончивших «Детский сад» и «Детский городок», показатели распределяются по вузам и формам обучения.

Всего для представителей этой категории по республике выделено 1580 дополнительных квот. По формам обучения:

  • по дневной форме обучения - 1137;
  • по заочной форме обучения - 377;
  • по вечерней форме обучения - 66.

Кроме того, утверждены дополнительные показатели приема детей сотрудников органов внутренних дел.
