Дания: частично финансируемая программа грантов для обучения на бакалавриате.

Объявлены важные новости для студентов и абитуриентов.

Резюме:
Кратко о грантовой программе: Ольборгский университет (AAU) в Дании объявляет о частично финансируемой грантовой программе под названием Nordplus Scholarships для выдающихся иностранных студентов для обучения на одном из университетских курсов бакалавриата в 2021-2022 учебном году.
University Times #201-250 в рейтинге международных университетов, опубликованном Higher Education (THE).
Преимущества стипендии: стипендии в размере 7000 датских крон (1110 долларов США) в семестр будут выделяться студентам, выигравшим стипендию.

Кратко о стипендиальной программе:

Ольборгский университет (AAU) в Дании, отличный международный университет, объявляет о частично финансируемой программе стипендий Nordplus для студентов, обучающихся на одном из университетских курсов бакалавриата в 2021-2022 учебном году. Университет занимает №201-250 в рейтинге международных университетов, опубликованном Times Higher Education (THE).

Преимущества гранта:

Стипендии в размере 7000 датских крон (1110 долларов США) в семестр будут выделяться студентам, выигравшим грант. Чтобы сохранить грант, студенты должны учиться с отличными оценками.

Требования:

  • Чтобы иметь возможность подать заявку на грант, кандидаты должны сначала подать заявку здесь на один из университетских курсов бакалавриата и быть принятыми;
  • После этого по этой ссылке можно будет подать заявку отдельно;
  • Кандидаты должны иметь один из следующих языковых сертификатов:
    • IELTS (академический тест + специальный онлайн-тест): общий балл 6,5;
    • TOEFL (интернет-версия + TOEFL iBT Special Home Edition): 85;
    • Квалификация C2 (CPE): 180/уровень C1;
    • C1 Advanced (CAE): 180/класс C;
    • ECPE: Пройден тест, включая сертификат

Примечание: С общими требованиями для поступления на бакалавриат университета можно ознакомиться по этой ссылке.
