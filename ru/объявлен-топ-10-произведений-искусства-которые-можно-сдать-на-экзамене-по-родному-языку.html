Объявлен ТОП-10 произведений искусства, которые можно сдать на экзамене по родному языку

Краткое содержание:
Агентство по оценке знаний и навыков опубликовало список ТОП-10 возможных произведений искусства для абитуриентов, сдающих экзамены по предмету родной язык и литература.
Вопросы по этим произведениям с высокой вероятностью возникнут на вступительных экзаменах.
Рекомендуемые ТОП-10 произведений искусства: Абдулла Кадыри - «Прошлые дни» Абдулла Кадыри - «Скорпион от алтаря» Откир Гашимов - «Между двумя дверями» Откир Гашимов - «Есть свет, но есть тень» Мирмухсин - «Зодчий» Чолпон - «Ночь и день» Абдурауф Фитрат - «Абдулфайзхан» Абдулла Орипов - «Мастер» Откир Гашимов - «Последняя жертва войны» Истории Абдуллы Каира (все) Почему этот список важен?

Агентство по оценке знаний и навыков объявило ТОП-10 возможных произведений искусства для абитуриентов, сдающих экзамен по родному языку и литературе. Высока вероятность, что вопросы по этим произведениям будут заданы на вступительных экзаменах.

Рекомендуемые ТОП-10 произведений искусства:

  1. Абдулла Кадири - «Прошлые дни»

  2. Абдулла Кадири - «Скорпион от алтаря»

  3. Шарп Хошимов - «Между двумя дверями»

    Фитрат

    - «Абдулфайзхан»

  4. Абдулла Орипов - «Сахибкирон»

  5. Откир Гашимов - «Последняя жертва война"

  6. Рассказы Абдуллы Кахора (все)

Почему этот список важен?

Эти произведения являются основными литературными источниками, изучаемыми в программе средней школы, и абитуриенты должны:

  • Содержание произведения,

  • Анализ изображений,

  • Идейные направления,

  • Обязательно глубокое знание стиля писателя.

Рекомендация соискателям:

Не следует ограничиваться только чтением произведений. Рекомендуется проанализировать:

  • Основные события,

  • Сюжетные линии,

  • Художественный стиль автора каждого произведения.
